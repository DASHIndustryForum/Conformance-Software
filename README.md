Conformance-Software
====================

This package, for now, only provides the Web-FE of MPEG-DASH/DASH-IF ISO-Segment Validator for Windows platforms.

Tested for:

1) Chrome for Window, Linux, and OS X.
2) IE.
3)Firefox
4)Safari for Windows
